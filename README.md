# Compass

## URL
[https://compass-93de8.firebaseapp.com/](https://compass-93de8.firebaseapp.com/)

## Features

- Nearby search
- Specific area search
- Place autocomplete
- Geocoding

## Used Framework & Libraries

- VueJS
- Vue Router
- Vuex
- Axios
- Firebase
- Kakao Map API
